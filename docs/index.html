Voice → text ⌥⇧R

Is it safe to run?

Klip is open source — you can read every line before you run it.

Does it phone home?

No telemetry, no analytics. Nothing leaves your Mac except the audio you explicitly send to OpenAI/Gemini — and transcription is on-device by default, so even that is opt-in.

Is the source auditable?

100% MIT-licensed on GitHub. It builds with the Xcode Command Line Tools — no opaque binaries, no hidden build step.

What permissions does it ask for?

Accessibility (auto-paste) and Screen Recording (capture) — requested once, only when you first use those features. Nothing else.

Is the build signed?

install.sh creates a local signing certificate so macOS remembers permissions across updates. Notarized distribution is on the roadmap.

Where does my data live?

Only on your Mac, in ~/Library/Application Support/Klip (0600). Credentials are AES-256-GCM encrypted at rest, key in the Keychain.

Is it really free?

Yes — free and open source under MIT. Use it, fork it, ship it.

Do I need an M1 / Apple Silicon Mac?

No. Klip runs on Intel and Apple Silicon Macs — the only requirement is macOS 14 (Sonoma)+. On-device transcription is faster on Apple Silicon's Neural Engine, but works on Intel too.

Does it work on Windows?

Not yet — Klip is macOS only for now. A Windows version is coming soon.

Can I contribute?

Please do. We're open to issues, pull requests and ideas on GitHub — feature requests and improvements welcome.

⬇ Download Klip.app

One-click app · or build from source below

# 1. Command Line Tools (if you don't have them)
xcode-select --install

# 2. Clone & install
git clone https://github.com/tamibot/klip.git klip
cd klip
./install.sh
1It builds & signs Klip, copies it to /Applications and launches it.
2The 📋 icon appears in the menu bar. Press ⌥⇧E to open the history.
3Grant Accessibility for auto-paste and Screen Recording on first capture — asked once.
4Optional: add an OpenAI/Gemini key for cloud transcription (stored in a local 0600 file).

Want to look first? build.sh makes a Klip.app without installing.
