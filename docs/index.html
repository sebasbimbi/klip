What it does

Everything you copy while coding with AI, in one menu-bar window.

🔎
Searchable history

Every copy — text & images — with instant search, type filters and keyboard navigation.

⭐️
Favorite & rename

Star the clips you reuse and give them names like “Deploy script” — find them by what they do.

✂️
Capture + annotate

Snip a screen region, mark it up with arrows/text, and it lands in your history.

🔤
Fast OCR

Snip an area and its text goes straight to the clipboard — pull a log out of a screenshot.

🎙️
Voice → text

Dictate a prompt — or drop an audio/video file — and it transcribes on-device (no key, nothing uploaded).

🧹
Paste clean / for chat

Paste as a clean code block, or reformat for WhatsApp / email — bold & italic kept.

🔐
Encrypted credentials

API keys are auto-detected, masked, and encrypted at rest (AES-256-GCM, key in the Keychain).

🌐
8 languages

Interface in English, Spanish, French, German, Italian, Portuguese, Chinese & Japanese.

Shortcuts

⌥⇧ + a letter, grouped by function — fully configurable.

⌥⇧EOpen the clipboard history
⌥⇧RRecord / stop a voice note
⌥⇧DCapture a region & annotate
⌥⇧FFast text capture → OCR to clipboard
⌥⇧OUpload an audio or video file to transcribe
⌘↩Copy the selected clip as a code block

Install

Build from source with just the Xcode Command Line Tools — no full Xcode.

# Command Line Tools (if you don't have them)
xcode-select --install

# Clone & install
git clone https://github.com/tamibot/klip.git klip
cd klip
./install.sh
1It builds & installs Klip to /Applications and launches it.
2The 📋 icon appears in the menu bar — press ⌥⇧E to open it.
3Grant Accessibility (auto-paste) and Screen Recording (capture) when asked — once.

🖥️ Currently macOS only (14 Sonoma+, Intel or Apple Silicon). A Windows version is coming soon. A prebuilt download will land on the GitHub releases page.

FAQ

Open source — read every line before you run it.

Do I need an M1 / Apple Silicon Mac?

No. Klip runs on Intel and Apple Silicon — the only requirement is macOS 14 (Sonoma)+. On-device transcription is faster on Apple Silicon, but works on Intel.

Does it work on Windows?

Not yet — macOS only for now. A Windows version is coming soon.

Is it private?

No telemetry, nothing tracked. Clips stay on your Mac; transcription is on-device by default. Cloud transcription (OpenAI/Gemini) is opt-in.

Is it safe to run?

100% MIT open source — auditable on GitHub. It only asks for Accessibility, Screen Recording and the Microphone — and only when you first use those features.

Is it free?

Yes — free and open source under MIT.

Can I contribute?

Please do — we're open to issues, pull requests and ideas on GitHub.
