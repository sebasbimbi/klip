📋 Klip

The clipboard manager for vibe coders — native to Mac.

Everything you copy while building with AI — code, errors, screenshots, prompts and API keys — one shortcut away. Lives in the menu bar: light, fast and private.

⚡ Install ★ View on GitHub
Swift · no Electron · runs offline · your data stays on your Mac

Snip an area → it lands in Klip → pull the text (OCR) · and record a voice note that transcribes itself.

🖥️ Mac only for now — requires macOS 14 (Sonoma)+ (Apple Silicon or Intel). A Windows version is planned.

Why Klip if you code with AI?

“Vibe coding” is a constant back-and-forth of copy-paste between your editor and Claude, ChatGPT or Cursor. Klip is built for exactly that flow.

Never lose a snippet

Everything you copy lands in a searchable history — code, errors, terminal output, prompts.

Snip & annotate an error

Grab a region of the screen, mark it up with arrows/text, and it's in Klip ready to paste into the AI.

Pull text out of a screenshot

Fast OCR turns a region into copyable text — paste a log that was stuck in an image.

Dictate your prompts

Record a voice note; it transcribes on-device (no key, no upload) and lands as text.

Paste clean, anywhere

Copy as a code block for the AI, or reformat for WhatsApp / email in one click.

Keep your keys safe

API keys are detected, encrypted at rest, masked, and never auto-pasted.

Everything in one menu-bar app

No accounts, no cloud lock-in, no Electron. Just Swift.

📋

Clipboard history

Text & images, instant search with highlighting, keyboard navigation, and type filters (text · links · images · voice · credentials · favorites).

📸

Capture + annotate ⌥⇧D

Snip a region (ScreenCaptureKit), then arrows, text, highlighter, shapes & undo. Lands in history, ready for OCR.

🔎

Fast OCR capture ⌥⇧F

Snip a region and its text goes straight to the clipboard — skip the editor when you just need the text.

🎙️

Voice → text ⌥⇧R

On-device by default (WhisperKit — no key, nothing leaves your Mac), or OpenAI / Gemini. Upload audio files too ⌥⇧O, with a per-file language.

🧰

Built for pasting into AI

Copy as a code block ⌘↩ with a detected language, or copy for WhatsApp / for email (clean formatting, kept bold/italic & spacing).

✨

Always paste clean

A copy from a dark-themed AI chat is stored as clean text that keeps bold/italic + emojis but drops the background, colours and fonts.

🔑

Encrypted credentials

Tokens & API keys are auto-detected and encrypted at rest (AES-256-GCM, key in the Keychain) — never in items.json or backups in the clear.

📦

Bundle context

Multi-select clips and combine them into a PDF or ZIP to upload a whole context to the AI in one shot. Group into collections.

🌍

8 languages

Interface in English, Spanish, French, German, Italian, Portuguese, Chinese & Japanese.

One comfortable shortcut family

⌥⇧ + a letter, grouped by function on the left of the keyboard — and fully configurable.

⌥⇧EOpen the history panel
⌥⇧RRecord / stop a voice note
⌥⇧DCapture a region & annotate (draw)
⌥⇧FFast text capture → OCR to clipboard
⌥⇧OOpen the “upload audio” window
⌘↩Copy the selected item as a code block

Private by design

Your history never leaves your Mac unless you choose a cloud transcriber.

💻
Local-first

History, images & audio stay on your Mac with 0600 permissions.

🔐
Encrypted keys

Credentials AES-256-GCM encrypted at rest; the key lives in the Keychain.

📡
Offline transcription

On-device Whisper by default — no API key, no upload.

🚫
No telemetry

Nothing is tracked. Ignores password-manager content; exclude any app.

⚡ Install in 30 seconds

Builds with just the Xcode Command Line Tools — no full Xcode, no Homebrew.

# 1. Command Line Tools (if you don't have them)
xcode-select --install

# 2. Clone & install
git clone https://github.com/tamibot/klip.git klip
cd klip
./install.sh
1It builds & signs Klip, copies it to /Applications and launches it.
2The 📋 icon appears in the menu bar. Press ⌥⇧E to open the history.
3Grant Accessibility for auto-paste and Screen Recording on first capture — asked once.
4Optional: add an OpenAI/Gemini key for cloud transcription (stored in a local 0600 file).

Want to look first? build.sh makes a Klip.app without installing.
